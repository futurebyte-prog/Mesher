# 📖 Overview

Mesher is an open-source peer-to-peer communication platform focused on privacy, security, and user ownership.

Unlike traditional messaging platforms, Mesher aims to establish direct communication between devices using WebRTC, reducing reliance on centralized messaging infrastructure.

Our philosophy is simple:

<b>Your conversations should belong to you—not a server.</b>

# 🏗️ Architecture

Mesher is designed to minimize dependency on centralized infrastructure.

Signaling servers may be used only to establish peer connections. User messages remain encrypted and are never stored on a central messaging server.

---

# 🔒 Security Goals

- End-to-End Encryption
- Forward Secrecy (planned)
- Secure Key Exchange
- Authenticated Encryption
- Secure Random Number Generation
- Open Security Design

Mesher will rely on well-established cryptographic libraries rather than custom cryptographic algorithms.
